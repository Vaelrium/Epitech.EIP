En quelques mots:
\begin{itemize}
  \item Un serveur de synchronisation de fichiers
  \item Plusieurs services utilisables (Dropbox, Ubuntu One, GDrive, Amazon S3, SSH, FTP…)
  \item Configurable grâce à des règles
\end{itemize}
